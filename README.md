# 🍎 The Fruit Store

A vibrant and responsive e-commerce front-end built to showcase fresh produce with a modern UI. This project focuses on high-performance styling and a seamless user experience.

## 🚀 Features

* **Dynamic Fruit Display:** Interactive menu components to browse various fruit selections.
* **Modern UI/UX:** Styled with Tailwind CSS for a sleek, professional look.
* **Fully Responsive:** Optimized for all screen sizes, from mobile to desktop.
* **Custom Footer:** Recently finalized detailed footer with navigation links and social integration.

## 🛠️ Tech Stack

* **Frontend:** React.js
* **Styling:** Tailwind CSS, PostCSS
* **Icons/Assets:** Lucide React (or your preferred icon library)
* **Deployment:** [e.g., Vercel / GitHub Pages]

## 📂 Project Structure

```text
├── src
│   ├── components
│   │   ├── Menus.jsx       # Fruit data display logic
│   │   ├── Navbar.jsx      # Navigation links
│   │   └── Footer.jsx      # Completed footer component
│   ├── assets              # Images and static files
│   └── App.js              # Main application entry
├── tailwind.config.js      # Custom styling configurations
└── package.json            # Project dependencies

```

## ⚙️ Getting Started

1. **Clone the repository:**
```bash
git clone https://github.com/HAMYL-Aththnayaka/The-Fruit-Store.git

```


2. **Install dependencies:**
```bash
npm install

```


3. **Run the development server:**
```bash
npm run dev

```
